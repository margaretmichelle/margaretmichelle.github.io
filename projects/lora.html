Overview

LoRa is a modulation technique used to encode information on radio waves for long-range applications highly robust to noise. We are designing a system with LoRa, that goes on your cat's collar, in a watertight enclosure, and provides live tracking updates to a custom mobile app using a GPS module. Through the use of an IMU and custom firmware, we are creating a low-power embedded system that can last for days before needing to be recharged.
