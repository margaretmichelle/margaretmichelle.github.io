Overview

During the summer after the second year of Engineering Physics, there is a robot competition; our year's theme was treasure hunting. An obstacle course was created with a variety of challenges and idols to steal. (Full details can be found here.) My team of 4 designed and built a fully autonomous, wheeled robot that made it to the quarter finals and competed in an instant death match against the winning team. We modeled, laser cut, 3D printed, and assembled the chassis; designed, soldered, and tested all our circuits; developed, integrated, tested, and optimized our C++ code until we had a robot that could consistently line follow and retrieve idols.



Robot Functions

  • Line following using IR photoelectric sensors and PID
  • Object detection using ultrasonic sensors
  • Object pickup, including self-positioning and dropping into storage basket
  • Magnet detection using Hall effect sensors
  • Bridge deployment to cross a chasm


My Contributions

  • Designed claw in Onshape and laser cut from acrylic sheets
  • Organized GPIO pin assignment for two STM32 microcontrollers (acting as master and slave) connected to the motors, sensors, and power
  • Programmed object detection and claw operation
  • Assisted in programing navigation system including interrupt handlers and PID control
  • Created a UI to see real time sensor data and modify PID coefficients to facilitate and expedite testing
  • Designed and soldered circuits for motor driving and photoelectric sensor signal filtering
  • Conducted full system testing
